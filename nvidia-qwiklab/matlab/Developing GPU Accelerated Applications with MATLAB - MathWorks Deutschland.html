Developing GPU Accelerated Applications with MATLAB

Learn How to Run MATLAB Code on NVIDIA GPUs in This Hands-On Lab

This lab demonstrates how you can leverage NVIDIA GPUs to accelerate computationally intensive MATLAB® applications in areas such as image processing, signal processing, and computational finance. You will learn about the different approaches to GPU computing in MATLAB, and see how MATLAB can be used to evaluate and test CUDA kernels.

Conclusions and Next Steps

These examples illustrate how to use MATLAB to develop GPU accelerated applications. Here are some best practices to keep in mind when performing GPU computing in MATLAB:

  • Profile your code to identify bottlenecks.
  • Work on large enough matrices to see the benefits of GPU parallelization.
  • Minimize data transfer between CPUs and GPUs by including a sustained use of supported functionality and creating variables directly on the GPU.
  • Use array indexing and branching in moderation.
  • Combine multiple element-wise calculations together into a single function call by using arrayfun.
